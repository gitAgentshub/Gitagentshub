# Contributing to Git Agents

Thanks for taking the time to contribute! 🎉

## How to Contribute

1. **Fork** this repository
2. **Create** your branch: `git checkout -b feat/your-feature`
3. **Commit** your changes: `git commit -m 'feat: add your feature'`
4. **Push** to your branch: `git push origin feat/your-feature`
5. **Open a Pull Request**

## Reporting Bugs

Found a bug? [Open an issue](../../issues/new?template=bug_report.md) and include:

- Steps to reproduce
- Expected vs actual behavior
- Browser/OS if relevant
- Screenshots if applicable

## Suggesting Features

Have an idea? [Open a feature request](../../issues/new?template=feature_request.md) describing the problem it solves and how you imagine it working.

## Code Style

- Keep the project dependency-free where possible (plain HTML/CSS/JS)
- Match the existing formatting and naming conventions in `index.html`
- Test changes in a browser before submitting a PR

## Code of Conduct

This project follows a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.
